Campus Information Website
Project Overview

This project is a single-page Campus Information Website designed to present important campus locations, student services, and announcements in a clear and organized way. The website demonstrates the use of HTML and CSS to structure content, control layout, and manage element positioning and visibility.

Features

A navigation bar with internal links and a dropdown menu for easy navigation

An introduction section explaining the purpose of the website and the type of campus information available

A Campus Services section using card layouts to display student support services

A Campus Locations/Gallery section arranged in a multi-column layout with images

An Announcements & Notices section featuring highlighted notices, including a sticky notice that remains visible while scrolling

A Display Behavior section showing visible and hidden elements to demonstrate CSS display properties

Technologies Used

HTML5 for semantic structure and content organization

CSS3 for layout, styling, spacing, positioning, and visual consistency

File Structure

index.html – Contains the website structure and content

style.css – Contains all styling and layout rules

Purpose of the Project

The main goal of this project is to demonstrate an understanding of CSS layout techniques such as flexbox, grid, positioning, and display behavior while creating a realistic and user-friendly campus website design.
